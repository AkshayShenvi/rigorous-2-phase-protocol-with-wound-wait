check requirements.txt for requirements
To install the required packages, run the following command:

pip install -r requirements.txt


To run the program, run the following commands:

python project1.py <name of input file in the same folder>


You will get an output file with the name output_of_<name of input file>

eg.
if you run:
python project1.txt input1.txt

you will get:
output_of_input1.txt
